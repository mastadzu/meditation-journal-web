# Meditation Journal Web

Веб-версия MVP для теста в браузере.

## Запуск
1. Открой терминал в `/Users/russell_maze/Documents/Codex/meditation-journal-web`.
2. Запусти локальный сервер:
   - `python3 -m http.server 4173`
3. Открой в браузере: `http://localhost:4173`

### Быстрый запуск на macOS
- Двойной клик по файлу:
  - `/Users/russell_maze/Documents/Codex/Open Meditation Journal Web.command`
- Он поднимет сервер и откроет:
  - `http://127.0.0.1:8787`

### Если VPN мешает `localhost`
- Двойной клик по файлу:
  - `/Users/russell_maze/Documents/Codex/Open Meditation Journal Web (No Server).command`
- Откроется версия без сервера через `file://`

### Остановка сервера
- `pkill -f \"python3 -m http.server 8787\"`

## Что реализовано
- Таймер медитации (1-360 минут) с автозачетом по завершению.
- Счетчики: медитации, намерения, инсайты, идеи.
- Журнал дня: отдельные записи по трем типам.
- Архив по месяцам/годам.
- Напоминания: любое количество, время, текст, вкл/выкл, удаление.
- PWA-режим (можно установить на устройство).

## Ограничение браузера
Фоновые напоминания работают корректно, когда вкладка открыта. Полностью фоновые уведомления без открытой вкладки требуют серверный push.
